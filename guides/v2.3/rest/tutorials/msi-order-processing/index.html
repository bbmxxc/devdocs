Order Processing with Multi Source Inventory

Before you begin

This tutorial builds upon the workflow described in the Order Processing with REST APIs tutorial. The Order Processing with Multi Source Inventory (MSI) tutorial provides additional steps that detail how to create stocks and sources, assign products to a custom source, and run the Source Selection Algorithm to recommend shipping options.

For more information about MSI, see the MSI wiki.

This 13-step tutorial generally takes 1 hour.

The Order Processing with MSI tutorial is a draft. It will be expanded to include more steps that show MSI functionality. This tutorial does not currently use the Luma sample store, but later versions will.

Complete these prerequisites

  • Install the latest version of Magento 2.3 Alpha from the MSI repository. Additional information is available at How to get the Magento software.

  • Install a REST client. You can use any REST client to send calls to Magento. Postman is recommended.

  • Know how to construct a REST call in Magento. See Construct a request for details.

  • Obtain an admin authorization token. Multiple calls in this tutorial require administrator privileges. See Generate the admin token for more information.

Other resources

  • Magento uses Swagger to provide REST API documentation. You can view the static REST API documentation or generate a local API reference.
